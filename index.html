üêß Linux & Shell Scripting Basics

Master the command-line fundamentals essential for bioinformatics analysis. These skills form the foundation of efficient RNA-seq data processing.

Essential Commands

File Navigation
Basic
Navigate directories, list files, and understand file paths. Master pwd, cd, ls, and file globbing patterns.
cd /data/rnaseq/ ls -lh *.fastq.gz pwd
File Operations
Basic
Copy, move, and manipulate files. Learn cp, mv, rm, and wildcard usage for batch operations.
cp sample*.fastq backup/ mv old_results/ archive/ rm temp_*.txt
Text Processing
Intermediate
Process text files with grep, sed, awk. Essential for parsing log files and extracting QC metrics.
grep "^@" sample.fastq | wc -l awk '{sum+=$3} END {print sum/NR}' \ counts.txt
Loops & Automation
Advanced
Automate repetitive tasks with for loops and bash scripts. Process multiple samples efficiently.
for file in *.fastq; do echo "Processing $file" fastqc $file done

Useful Bash Scripting Patterns

#!/bin/bash # Process all RNA-seq samples SAMPLES="sample1 sample2 sample3" REF_GENOME="genome.fa" for sample in $SAMPLES; do echo "Starting analysis for $sample" # Run QC fastqc ${sample}_R1.fastq.gz ${sample}_R2.fastq.gz # Alignment STAR --genomeDir $REF_GENOME \ --readFilesIn ${sample}_R1.fastq.gz ${sample}_R2.fastq.gz \ --readFilesCommand zcat \ --outFileNamePrefix ${sample}_ echo "Completed $sample" done echo "All samples processed!"

Quality Control Tools

FastQC
Essential
Comprehensive quality assessment of high-throughput sequencing data. Generates detailed HTML reports with multiple QC metrics.
Bioconductor GitHub Paper
# Load libraries library(DESeq2) library(ggplot2) # Read count matrix countData <- read.table("counts.txt", header=TRUE, row.names=1) # Sample information colData <- data.frame( condition = factor(c("control", "control", "control", "treated", "treated", "treated")), row.names = colnames(countData) ) # Create DESeq2 dataset dds <- DESeqDataSetFromMatrix( countData = countData, colData = colData, design = ~ condition ) # Filter low count genes keep <- rowSums(counts(dds)) >= 10 dds <- dds[keep,] # Run DESeq2 analysis dds <- DESeq(dds) # Extract results res <- results(dds, contrast=c("condition","treated","control")) # Shrink log2 fold changes resLFC <- lfcShrink(dds, coef="condition_treated_vs_control", type="apeglm") # Summary summary(res) # Order by p-value resOrdered <- res[order(res$pvalue),] # Save results write.csv(as.data.frame(resOrdered), file="DESeq2_results.csv")

Best Practices & Guidelines

Key Recommendations

  • Experimental Design: Include biological replicates (minimum 3 per condition), consider batch effects, and randomize sample processing
  • Quality Control: Always run FastQC before and after trimming, check for adapter contamination, and monitor alignment rates
  • Alignment: Use STAR for splice-aware alignment, include gene annotations for better junction detection
  • Quantification: For gene-level analysis use featureCounts or HTSeq; for transcript-level use Salmon/Kallisto + tximport
  • Differential Expression: Filter low-count genes, use appropriate normalization, and apply FDR correction (padj < 0.05)
  • Visualization: Create PCA plots to check sample clustering, MA plots for DE assessment, and heatmaps for top genes
  • Validation: Validate key findings with qRT-PCR, examine individual gene plots, and perform functional enrichment analysis

Alignment Tools

STAR
Recommended
Spliced Transcripts Alignment to a Reference. Ultra-fast RNA-seq aligner optimized for detecting splice junctions. Gold standard for RNA-seq.
GitHub Manual
# Index genome STAR --runMode genomeGenerate \ --genomeDir genome_index/ \ --genomeFastaFiles genome.fa \ --sjdbGTFfile genes.gtf \ --runThreadN 8 # Align reads STAR --genomeDir genome_index/ \ --readFilesIn R1.fastq.gz R2.fastq.gz \ --readFilesCommand zcat \ --outFileNamePrefix sample_ \ --outSAMtype BAM SortedByCoordinate \ --runThreadN 8

üìä Read Count Generation

Quantify gene expression levels by counting reads aligned to genomic features. Choose between feature-based counting or transcript-level quantification.

featureCounts
Recommended
Fast and efficient read counting tool from the Subread package. Highly accurate with excellent speed for both single-end and paired-end data.
Website Paper
featureCounts -a genes.gtf \ -o counts.txt \ -T 8 -p -B -C \ sample1.bam sample2.bam sample3.bam # -p: paired-end # -B: both ends mapped # -C: exclude chimeric fragments # -T: threads
HTSeq-count
Classic
Widely-used Python tool for counting reads in genomic features. Offers flexible counting modes (union, intersection-strict, intersection-nonempty).
Docs GitHub
htseq-count -f bam -r pos \ -s no -t exon -i gene_id \ -m union \ sorted.bam genes.gtf \ > counts.txt # -s: stranded (no/yes/reverse) # -m: mode (union/intersection) # -t: feature type # -i: attribute ID
Salmon
Alignment-Free
Ultra-fast transcript-level quantification without alignment. Uses k-mer based approach. Great for transcript-level analysis and isoform detection.
Docs GitHub
# Index transcriptome salmon index -t transcripts.fa \ -i salmon_index # Quantify salmon quant -i salmon_index \ -l A -1 R1.fastq.gz -2 R2.fastq.gz \ -o salmon_output -p 8
Kallisto
Alignment-Free
Extremely fast pseudoalignment-based quantification. Requires minimal memory and is perfect for large-scale RNA-seq projects.
Website GitHub
# Build index kallisto index -i index.idx \ transcripts.fa # Quantify kallisto quant -i index.idx \ -o output -b 100 \ R1.fastq.gz R2.fastq.gz
RSEM
Isoform-Aware
RNA-Seq by Expectation-Maximization. Accurately quantifies transcript abundances accounting for uncertainty in read mapping to isoforms.
Website GitHub
# Prepare reference rsem-prepare-reference \ --gtf genes.gtf \ genome.fa ref/genome # Calculate expression rsem-calculate-expression \ -p 8 --paired-end \ R1.fastq.gz R2.fastq.gz \ ref/genome output
tximport
R Package
Import transcript-level estimates from Salmon/Kallisto into R and summarize to gene-level for DESeq2. Essential bridge between quantifiers and DE analysis.
Bioconductor
# R code library(tximport) library(readr) files <- list.files("salmon_output", pattern = "quant.sf", full.names = TRUE) txi <- tximport(files, type = "salmon", tx2gene = tx2gene)

DESeq2
Essential
Differential gene expression analysis based on the negative binomial distribution. Handles library size normalization, dispersion estimation, and statistical testing with sophisticated shrinkage methods.
